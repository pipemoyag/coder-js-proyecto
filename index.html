Simulador de tienda online: Cat Republic

Este simulador representa una tienda online especializada en productos para gatos. Al ingresar, el usuario puede explorar un catálogo de productos organizado por categorías como alimentos, snacks, juguetes y accesorios. Cada producto muestra su nombre, precio, y la cantidad que ya se encuentra en el carrito.

El catálogo ahora se carga dinámicamente desde un archivo JSON, utilizando la librería Axios. Para mejorar la experiencia del usuario se incorporaron notificaciones con Toastify y alertas interactivas con SweetAlert2.

Se implementó lógica de gestión de stock, que valida las cantidades tanto en la página de productos como en el carrito, evitando agregar más unidades que las disponibles.

Además, se reemplazaron los antiguos manejadores onclick y onchange por delegación de eventos, lo que permite un código más limpio, escalable y fácil de mantener.

El carrito se mantiene entre sesiones gracias a localStorage y permite actualizar cantidades, eliminar productos o vaciarlo completamente. Al momento de pagar, el usuario puede ingresar sus datos y confirmar la compra mediante un formulario interactivo.
